# mini-cursor

A pure Python project.

## Setup

1. Create a virtual environment:
   ```bash
   python3 -m venv venv
   ```

2. Activate the virtual environment:
   ```bash
   source venv/bin/activate
   ```

3. Install the project in editable mode:
   ```bash
   pip install -e .
   ```

## Running the app

```bash
python3 src/mini_cursor/main.py
```
